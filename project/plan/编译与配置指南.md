## sdkconfig.defaults

创建 `sdkconfig.defaults` 用于预设关键配置：

```
# Camera
CONFIG_ESP_VIDEO_ENABLE=n          # 暂不启用 esp-video 框架

# ISP
CONFIG_ISP_USE_ISP=y               # 启用片上ISP
CONFIG_ISP_ALGORITHMS_RGB_GAIN=n   # 简化ISP配置

# PSRAM
CONFIG_SPIRAM=y
CONFIG_SPIRAM_MODE_OCT=y           # 使用Octal PSRAM

# LVGL
CONFIG_LV_COLOR_DEPTH=16           # RGB565

# FreeRTOS
CONFIG_FREERTOS_HZ=1000            # 1kHz tick for physics timing

# Optimization
CONFIG_COMPILER_OPTIMIZATION_LEVEL_DEBUG=y   # 开发阶段 -Og

# Partition table for model storage
CONFIG_PARTITION_TABLE_CUSTOM=y
CONFIG_PARTITION_TABLE_CUSTOM_FILENAME="partitions.csv"
```

## 分区表

创建 `partitions.csv`（如果使用 SPIFFS 存储模型）：

```
# Name,   Type, SubType,  Offset,   Size,    Flags
nvs,      data, nvs,      0x9000,   0x6000,
phy_init, data, phy,      0xf000,   0x1000,
factory,  app,  factory,  0x10000,  0x400000,
storage,  data, spiffs,   0x410000, 0x300000,
```

模型文件（.espdl）放入 storage 分区。

## 编译命令

### 激活 ESP-IDF 环境

在 bash 中：
```bash
. "E:/ProgramDate/.espressif/v5.5.4/esp-idf/export.sh"
```

### 构建步骤

```bash
# 1. 设置目标芯片（首次）
idf.py set-target esp32p4

# 2. 配置（可选，使用默认配置即可）
idf.py menuconfig

# 3. 编译
idf.py build

# 4. 烧录（用户操作）
idf.py -p <COM_PORT> flash

# 5. 查看日志
idf.py -p <COM_PORT> monitor
```

简化编译命令（在项目目录下）：
```bash
"E:/ProgramDate/.espressif/v5.5.4/esp-idf/tools/idf.py" build
```

## 常见编译问题

1. **组件未找到**：确保 `managed_components` 目录下有对应组件，删除后重新 `idf.py build` 会自动下载

2. **PSRAM 不足**：检查 `CONFIG_SPIRAM` 是否开启，帧缓冲区使用 PSRAM 分配

3. **引脚冲突**：CSI 使用固定引脚，DVP 可配置。CSI引脚与SD卡部分共享（42-48），注意避免冲突

4. **Cache对齐**：大缓冲区分配时使用 `heap_caps_aligned_calloc` 确保 cache line 对齐
