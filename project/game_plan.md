# 游戏设计

### 综述

每个元素都是特定的对象，如墙用同一种像素组合表示，水果用一种特定的图表达（检测到apple，orange等就在游戏界面用这同一种图显示），即游戏界面是在原始获取信息的抽象。

游戏界面参考经典像素游戏，简洁而美观

### 各个元素特性

除物体外的边缘部分（边缘检测得到的边缘）是坚硬的墙，可反弹，无法穿过

各个物体是具有特殊属性的道具和障碍，考虑到其实际大小比较大，可用特定物体对应某个缩小版的东西

水果：拾取道具，得到后可获得一次穿墙能力

mouse：穿梭洞，对应于不同的鼠标位置（两个）

scissors：死亡地，触之即死

book：其构成的墙（书的边缘）可通过反复撞击破坏

bottle：终点

cup、spoon、keyboard、cell phone：构成不同弹力系数的区块
