# 游戏设计

### 综述

每个元素都是特定的对象，如墙用同一种像素组合表示，水果用一种特定的图表达（检测到apple，orange等就在游戏界面用这同一种图显示），即游戏界面是在原始获取信息的抽象。

游戏界面参考经典像素游戏，简洁而美观

### 各个元素特性

除物体外的边缘部分（边缘检测得到的边缘）是坚硬的墙，可反弹，无法穿过

各个物体是具有特殊属性的道具和障碍，考虑到其实际大小比较大，可用特定物体对应某个缩小版的东西

水果：拾取道具，得到后获得5秒穿墙能力（可穿过所有墙壁），+100分

mouse：穿梭洞，成对出现，弹珠进入一个后从另一个传送出来（方向随机），冷却1秒，+25分

scissors：死亡地，触之即死，扣除一条命，进入重生状态（800ms延迟后回到初始位置）

book：可破坏墙壁，需撞击3次（撞击速度需>500px/s），每次撞击后退化一级（3→2→1→碎），撞碎+50分

bottle：终点，触之即获胜，+500分 + 每秒剩余时间+10分

cup：捕获弹射 —— 弹珠进入杯子范围后被捕获冻结，玩家倾斜设备瞄准方向（画面显示金色箭头），1秒后沿瞄准方向弹射（初速600px/s），杯子冷却2秒

spoon：低弹力墙壁 —— 检测到勺子后，其区域转换为石墙瓦片，不可破坏，弹珠撞击时弹力系数仅0.15（几乎粘住）

keyboard：中等弹力表面（0.40），弹珠经过时反弹减弱

cell phone：高弹力表面（0.85），弹珠经过时反弹增强（接近蹦床）
